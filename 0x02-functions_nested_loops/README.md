# Holberton School - Low Level Programming
## 0x02 - C (Functions, nested loops)

### Purpose
This introductory C programming language project covers
* nested loops
* functions
* variable declarations and function definitions
* header files, and `#include` guard

### Requirements
* Ubunto 14.04 LTS
* gcc 4.8.4
